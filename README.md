User manual:
___________________________________________________________
Compile client.c and server.c (gcc filename.c -o filename)
Run both files on different terminal/pc (server.c must be run before client.c)
For example:

Terminal 1: ./server portNumber
Example: ./server 8888
Terminal 2: ./client IPaddress portNumber
Example: ./client 127.0.0.1 8888
